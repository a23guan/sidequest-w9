## GBDA302 Sidequest Week 9: Adding a debug panel

---

## Description

This project lets users access a debug panel by pressing the tilde (`) key. On the debug panel, users can toggle on/off mood gravity, hit boxes, and velocity/state stats of the player for easy debugging

---

## Interaction Instructions

Press the tilde (`) key to access debug panel. Press g/G for moon gravity, h/H for hit boxes, and v/V for velocity stats
